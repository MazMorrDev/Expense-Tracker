# Expense Tracker

This is MazMorrDev's solution to the [Expense Tracker project](https://roadmap.sh/projects/expense-tracker) from [roadmap.sh](https://roadmap.sh).  
I built my own version in C# as a command‑line tool to keep track of personal expenses.I hope you enjoy it :D

## 🚀 Features

- Add expenses with a description, amount, and optional category
- List all recorded expenses
- (More features coming soon – like summaries, filters, or exporting)
-

## 📦 Requirements

- [.NET 10 SDK](https://dotnet.microsoft.com/) or later

## 🛠️ Installation & Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/MazMorrDev/Expense-Tracker.git
   cd Expense-Tracker
   ```

2. Clean and Build project:

   ```bash
   dotnet clean
   dotnet build
   ```

3. Run the project

   ```bash
   dotnet run
   dotnet run add --description "Expense Description" --amount 678.32 --category "Your Expense Category"
   ```
