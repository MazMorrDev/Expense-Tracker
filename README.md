# Expense Tracker

This is my solution to the [Expense Tracker project](https://roadmap.sh/projects/expense-tracker) from [roadmap.sh](https://roadmap.sh).  
I built my own version in C# as a command‑line tool to keep track of personal expenses.I hope you enjoy it

## 🚀 Features (this section it's currently just a placeholder)

- Add expenses with a description, amount, and optional category
- List all recorded expenses
- (More features coming soon – like summaries, filters, or exporting)

## 📦 Requirements

- [.NET 10 SDK](https://dotnet.microsoft.com/) or later
